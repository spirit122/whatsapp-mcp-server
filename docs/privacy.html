Privacy Policy

Last updated: March 19, 2026

1. Introduction

WhatsApp MCP Server ("we", "our", "the Service") is operated by Spirit122 Studio. This policy describes how we handle information when you use our MCP Server for WhatsApp Business API integration.

2. Information We Collect

  • Account Information: Email address provided during subscription purchase via Lemonsqueezy.
  • API Credentials: WhatsApp Business API tokens you configure in the dashboard. These are stored encrypted and used only to make API calls on your behalf.
  • Usage Logs: Tool call names, timestamps, success/failure status, and response times for rate limiting and analytics. We do NOT store message content.
  • Webhook Data: Incoming WhatsApp messages are temporarily stored in Cloudflare Durable Objects for retrieval via MCP tools. Data is scoped per tenant and not shared.

3. How We Use Information

  • To provide and maintain the MCP Server service
  • To authenticate your API requests
  • To enforce rate limits and anti-spam protections
  • To generate usage analytics (Enterprise tier)
  • To process payments via Lemonsqueezy

4. Data Sharing

We do NOT sell, rent, or share your data with third parties. Your data is only shared with:

  • Meta/WhatsApp: API calls are made to Meta's WhatsApp Business Cloud API on your behalf.
  • Lemonsqueezy: Payment processing for subscriptions.
  • Cloudflare: Infrastructure provider (Workers, D1, KV, Durable Objects).

5. Data Storage and Security

  • All data is stored on Cloudflare's global edge network
  • API keys are hashed before storage
  • Webhook signatures are verified using timing-safe HMAC comparison
  • Tenant data is isolated via separate Durable Object instances
  • HTTPS/TLS encryption for all data in transit

6. Data Retention

  • Webhook messages: Retained for up to 1,000 messages per tenant, oldest deleted first
  • Usage logs: Retained for 90 days
  • Account data: Retained until subscription cancellation + 30 days

7. Your Rights

You can:

  • Delete your webhook data at any time via the MCP tools
  • Cancel your subscription to stop all data processing
  • Request data export or deletion by contacting us

8. Anti-Spam

We enforce anti-spam protections including recipient allowlists, per-recipient rate limits, unique recipient caps, and content filtering to prevent abuse of the WhatsApp Business API.

9. Contact

For privacy questions or data requests, open an issue on GitHub.
